# Advanced Career Guidance System

A Flask-based web application that provides personalized career recommendations using RIASEC personality assessment and semantic search.

## Features

- RIASEC personality assessment via direct scoring
- Advanced semantic search with multiple matching parameters
- Top 5 career recommendations with match percentages
- Modern, responsive UI
- Vector database for efficient similarity search
